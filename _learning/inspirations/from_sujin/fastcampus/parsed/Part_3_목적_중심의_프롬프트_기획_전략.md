실제 데이터를 기반으로 하므로 사용자의 실제 행동과 요구를 더 잘

용자들의 피드백을 분석하여 기능을 개선하고 발전
단점 데이터를 수집하고 분석하는 데 시간이 많이 걸림
수집된 데이터가 특정 사용자 그룹이나 특정 상황에 편향
거 데이터에 기반한 예측은 새로운 상황이나 트렌드 변화를 완전히

o 사용자 데이터 분석: 사용자들이 어떤 키워드나 질문을 주로 입력하는지 데이터 수집 및 분석
o 피드백 반영: 사용자 피드백을 바탕으로 프롬프트 기능을 개선하고 새로운 기능 추가
o A/B 테스트: 다양한 프롬프트 방식을 실험하여 가장 효과적인 방식을 찾기

연역적 방식 장단점

o 이론적 모델 수립: 사용자가 어떤 정보를 필요로 할지 이론적으로 분석하고 가설 세우기.

o 규칙 기반 시스템: 특정 규칙이나 알고리즘에 따라 프롬프트를 제공하는 시스템을 구축.
